사용 방법
====================================================================
1. LFS-Tools 폴더와 LFS-launcher.bat 파일을 프로젝트 파일로 복사합니다.
2. LFS-launcher.bat 을 실행 후
3. "[1] Auto-LFS-Tracker" 를 실행합니다.
4. "1. 대용량 파일 LFS 자동 설정" 을 선택하여 LFS 등록을 완료합니다.
5. LFS 등록이 완료되면 ".gitattributes 파일을 지금 커밋하시겠습니까?" 에서 y를 눌러 커밋을 진행합니다.
6. 이후 메인화면으로 돌아오면 "[2] LFS-Migration" 을 진행합니다.
7. 마이그레이션이 완료되면 Enter를 눌러 종료합니다.
	*현재 Enter를 눌러도 정상 종료가 되지 않는 문제가 있습니다. Enter를 눌러도 종료되지 않으면 콘솔창을 닫아도 됩니다.
8. LFS등록 및 마이그레이션이 완료되면 이후 GitHub Desktop을 실행하여 다시 Push를 진행하면 정상적으로 Push가 완료됩니다.
